						                       File Transfer Application

                             Introduction : This is an implementation of file transfer application made with the implementation of FTP using Java. 

                                        Note : No third party library is used except sockets as for the better understanding of the flow .

Methods to use  :

                 Here, two files is presented name FTServerFrame (for the server end) and FTClientFrame(for the client end).
                 Exceute FTServerFrame (java FTServerFrame) first and press start.

Start FTClientFrame (java FTClientFrame) and add some files to transfer and then press start.

                                                         Note : Must check for extra directory made after using this application.
